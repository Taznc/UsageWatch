UsageWatch / Compact Widget Theme Review

Pinboard stays. Four new compact directions join it.

This mockup pass keeps Pinboard Mini as the anchor and replaces the rejected concepts with a balanced set: a stacked monitor panel, a gauge-driven concept, a compact tile board, and a terminal-style deck. Every concept now uses provider header badges for Claude, Codex, and Cursor, matching the direction you want the live widget header to take on both macOS and Windows.

5 concepts total Styled provider badges Compact-first footprints Relevant text kept visible Balanced visual bundle
Live groundwork The widget header is being formalized around shared provider badges and typed theme customizations.
Mock data Claude is weekly-heavy, Codex has active credits, and Cursor is midway through the current cycle.
Review focus Footprint, scanability, and whether each concept earns a place as a future live theme family.
Coverage: Across the five concepts you can review `session`, `weekly`, `extra`, `balance`, `credits`, and muted `status` cards for the supported providers.
Compact rule: secondary copy stays short, graphics stay structural instead of decorative, and every header uses a provider badge instead of the older colored dot treatment.
